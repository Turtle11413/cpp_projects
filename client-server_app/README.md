# client-server_app

## Install
```bash
git clone git@github.com:Turtle11413/client-server_app.git
cd client-server_app
```

## OS
- Linux
- MacOS

## dependencies
- Qt6
- make

## Run server
```bash
cd server
make run_server
```

## Run client
In other terminal session !!!
```bash
cd client
make run_client
```
